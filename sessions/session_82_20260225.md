## Learnings

- **AT peer debate achieves autonomous convergence**: TC2 resolved Critic's DEFER→MODIFY without user intervention. TC1 required user challenge + 2 followup agents. Primary value of --exploratory: real-time adversarial debate prevents positions from becoming fixed before challenges arrive.
- **AT more token-efficient for adversarial topics**: 4 AT agents (~55%) < 5+2 sequential agents (~78%). Peer debate resolves disagreements inline rather than requiring followup cycles.
- **Sync script needs per-skill transform awareness**: Generic `bulwark-{name}` → `{name}` sed is insufficient. Skills reference each other (`bulwark-research` inside brainstorm), use capitalized forms in headers, and embed rule IDs (SA2). The transform loop now handles all these cases.
