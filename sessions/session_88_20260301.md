## Technical Decisions

### Distribution: npm Promoted to v1
- **Decision**: npm is a first-class plugin source in marketplace.json, not a separate packaging effort. Include as v1 alongside self-hosted GitHub marketplace.
- **Rationale**: Marketplace docs show npm as just another `source` type — trivial addition. Windows/WSL users need a non-homebrew path.
- **Impact**: Steps 12-17 in implementation outline; package.json already exists

### GitHub Org: Q-Ball-Inc
- **Decision**: All plugins (Bulwark, Clear Framework, future) hosted under Q-Ball-Inc org with single `qball-inc` marketplace
- **Rationale**: Single discovery point for users, independent repos per plugin, cross-promotion
- **Impact**: Repo transfer via GitHub Settings (preserves history + redirects). Timing: after P7.2 docs brainstorm

### CLAUDE.md Injection Descoped
- **Decision**: `init-project-rules.sh` removed from init pipeline. CLAUDE.md injection redundant with `.claude/rules/rules.md` auto-loading.
- **Rationale**: `lib/templates/claudemd-injection.md` injects Binding Contract + "Read Rules.md" — both already enforced by rules in `.claude/rules/`. No governance value added.
- **Impact**: init.sh only runs init-rules.sh (copies rules.md to `.claude/rules/`)

### Marketplace Name Independent of Repo Name
- **Decision**: Marketplace name `qball-inc` in marketplace.json, repo name `Q-Ball-Inc/claude-plugins`
- **Rationale**: Marketplace `name` field is just an identifier for install commands, doesn't need to match repo name
- **Impact**: Users run `/plugin marketplace add Q-Ball-Inc/claude-plugins` then `/plugin install the-bulwark@qball-inc`

## Learnings

- **npm is not a separate distribution architecture** — it's just another `source` type in marketplace.json. The synthesis incorrectly deferred it to "v3, if ever" by treating it as a packaging problem when it's a one-line config change.
- **Marketplace name is decoupled from repo name** — `name` field in marketplace.json is an install-command identifier, not tied to GitHub repo naming.
- **Official Anthropic marketplace has submission forms** — `claude.ai/settings/plugins/submit` and `platform.claude.com/plugins/submit`. Timeline is unknown/uncontrolled, so self-hosted marketplace is the v1 distribution path.
- **Plugin caching copies to ~/.claude/plugins/cache** — plugins can't reference files outside their directory (no `../`). Symlinks are followed during copy as a workaround.
