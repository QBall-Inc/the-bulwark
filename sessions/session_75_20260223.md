## Learnings

- **Sub-agents can invoke agents via headless mode**: `claude -p --agent <name>` via Bash tool. Discovered when 8 parallel sub-agents independently found this workaround for the "sub-agents can't use Task tool" constraint. Trade-offs (no structured return, no resume) are moot in Bulwark's log-based architecture. Partially resolves the Session 71 finding that pipeline agents are infeasible. Implications for P6.7 and P6.8. Full doc: `docs/headless-agent-invocation.md`.
- **Substitution compliance checklist works**: The Mandatory Execution Checklist restructuring from Session 74 was effective — all 8 validation sub-agents invoked anthropic-validator correctly (via headless fallback). The checklist-at-top pattern is validated.
- **Validator consistency varies by doc-fetch success**: Some sub-agents fetched live docs from code.claude.com, others fell back to cached/embedded checklists. This caused minor finding variance across agents (e.g., `argument-hint` flagged as medium in one run but not others). Not a pipeline bug — inherent in dynamic doc fetching.
