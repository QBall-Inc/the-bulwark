## Learnings

- **`init` not in official plugin schema**: The internal prepare-plugin-pkg reference had `init` as a valid field, but official Anthropic docs don't support it. Always verify against official docs for distribution-facing configuration.
- **`author` is an object, not a string**: Official schema specifies `{ name, email, url }`. Plain strings may work but don't match the spec.
- **All plugin.json paths must start with `./`**: Official convention. Bare relative paths like `"skills/"` should be `"./skills/"`.
- **`private: true` blocks npm publish**: The original package.json had it. Removal was intentional for S92 npm distribution.
