## Learnings

### Skill Dependency Chain
- `suggest-pipeline.sh` → `pipeline-templates` → `subagent-prompting` works for Write|Edit
- Direct skill invocation bypasses this chain
- Fix (P6.1): Add `subagent-prompting` to skills array of orchestration skills

### Orchestration vs Sub-Agent Skills
- Skills with `agent:` field run as sub-agents with specified model
- Orchestration skills (no `agent:`) run in main context and spawn sub-agents
- Don't confuse model for sub-agent spawning with model for skill execution
