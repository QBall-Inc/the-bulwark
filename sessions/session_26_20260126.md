## Learnings

### Hook Behavior Clarification
- SessionStart exit 0 stdout → Claude's CONTEXT (visible in conversation)
- PostToolUse exit 2 → blocks operation, stderr shown to Claude
- `once: true` resets on new session, not on /clear or /compact

### WSL Line Ending Issues
- Windows line endings (CRLF) break bash scripts
- Always run `sed -i 's/\r$//'` after creating scripts
- awk patterns need `\r?$` to match both LF and CRLF in source files
