## Learnings

### Checklist Enforcement for LLMs
- Single hard instructions get lost in context flood
- Checklists with required documentation create accountability at each step
- Making shortcuts visible (empty checklist entries) is more effective than prohibition

### Test Fixture Design
- Fixtures without package.json cause npm error 254
- Agent correctly falls back to generated JS scripts for validation
- This tests the fallback strategy path, which is valuable
